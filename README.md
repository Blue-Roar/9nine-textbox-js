# 9-nine- 魔裁对话框表情包生成器

一个9-nine-表情包生成工具，能够快速生成带有自定义文本的魔裁对话框图片。

修改自[魔法少女的魔女审判 文本框表情包生成器](https://github.com/Blue-Roar/manosaba-textbox-js)，该仓库是Python脚本[魔法少女的魔女裁判文本框脚本](https://github.com/oplivilqo/manosaba_text_box)的JavaScript移植与实现。

## 使用方法

无需部署到Web服务器，直接下载整个仓库然后本地使用浏览器打开`index.html`即可使用。
